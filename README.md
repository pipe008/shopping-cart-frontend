# 🛒 Tienda Central - Frontend

Esta es la interfaz de usuario de **Tienda Central**, un catálogo de compras en línea moderno, rápido y totalmente responsivo. El proyecto está construido siguiendo las mejores prácticas de desarrollo web para ofrecer una experiencia de usuario fluida y profesional.

## ✨ Características de la Interfaz

* **Catálogo Interactivo:** Visualización dinámica de productos con carga optimizada.
* **Búsqueda en Tiempo Real:** Filtrado de productos por nombre o descripción mediante una barra de búsqueda intuitiva.
* **Filtrado por Categorías:** Selector integrado para navegar fácilmente entre diferentes tipos de productos.
* **Navegación Fluida:** Sistema de rutas para secciones de Productos, Administración, Carrito, Checkout y Pedidos.
* **UI/UX Limpia:** Diseño minimalista con componentes estilizados, banners informativos y retroalimentación visual clara.

## 🚀 Tecnologías Utilizadas

* **React.js:** Biblioteca principal para la creación de componentes reutilizables.
* **Vite:** Herramienta de construcción (Build Tool) para un desarrollo ultrarrápido.
* **JavaScript (ES6+):** Lógica del lado del cliente.
* **React Router:** Gestión de navegación Single Page Application (SPA).
